# cookbook-app
cookbook is a mobile application that allows users to discover recipes from all around the world. The app organizes countries and recipes from A-Z making it easy to browse and explore recipes.
# Project Goals
Browse Countries from A-Z
Discover authentic recipes
Search recipe by country or name
Create a clean user experience
# Planned Tech Stack
Flutter
Firebase
Figma
Git & Github
